Notes
- XMP = Intel memory profiles; EXPO = AMD memory profiles.
- For RTX 4090 class GPUs, use ATX 3.0/3.1 PSUs with native 12VHPWR (see product_list.txt).
- If memory overclock is unstable, test at JEDEC base speed before RMA.
- Always update BIOS before CPU swaps to ensure post and microcode support.
